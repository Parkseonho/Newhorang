# 🐯 teamHORNG 익명 게시판 프로젝트
## 개발 환경
* VScode
* SQLyog
* GitHuB
## 사용 기술

### 백엔드

* Node.js

### DataBase

* Xampp
* MySql

### 프론트 엔드

* React
* Javascript
* tailwind

## Description
* React와 Node.js를 이용한 익명 게시판

## Main Service
* 익명게시판 서비스
* 비밀게시판 서비스
* 내글 보기 서비스
* 게시글 검색 서비스
